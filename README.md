# 🏙️ Brockton Public Opinion Analysis

A comprehensive data analysis and visualization dashboard built with **Python & Streamlit** to evaluate public opinion concerning the youth in Brockton, Massachusetts.

This project aggregates, synthesizes, and visualizes community voices from various social media and local news platforms to identify core concerns, sentiments, and trending topics regarding education, safety, and youth programs in the city.

---

## 🚀 Features

- **Interactive Streamlit Dashboard:** A premium, mathematical representation of public sentiment natively in Python.
- **Live News Scraping:** Integration with BeautifulSoup to scrape live local news updates (e.g., CBS Boston News) directly into the dashboard.
- **Advanced Filtering:** Dynamically slice the dataset by topic (Education, Safety, Parks) and sentiment.
- **AI Insights:** Automated extraction of prevailing tones and keywords using NLTK/TextBlob.

## 📖 Methodology & Data Sources

To capture a diverse range of community voices, this project evaluates the following platforms representing different demographic perspectives:

1. **Facebook (Adult & Institutional):** Includes pages like *The Brockton Hub* and *Brockton Public Schools Official* to capture civic concerns and educational updates.
2. **Twitter/X & Local News (Real-Time):** Leverages local journalism (e.g., *CBS Boston News*, *The Enterprise News*) and keyword streams for unfiltered reactions.
3. **Instagram (Youth Perspective):** Focuses on school organizations and athletic pages to represent the direct, lived experience of the youth.

*Note: Due to strict modern anti-scraping protections on social media, the dashboard relies on a combination of live local news scraping and a robust Synthetic Dataset Generator (`generate_mock_data.py`) to guarantee a functional analytical demonstration.*

## 💻 How to Run Locally

To run the analysis dashboard on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MalikZeeshan1122/brockton-public-opinion-analysis.git
   cd brockton-public-opinion-analysis
   ```

2. **Install dependencies:**
   Make sure you have Python installed, then run:
   ```bash
   pip install streamlit pandas plotly wordcloud matplotlib beautifulsoup4 requests textblob
   ```

3. **(Optional) Generate Mock Data:**
   If the dataset is missing, generate the synthetic data base first:
   ```bash
   python data/generate_mock_data.py
   ```

4. **Run the application:**
   Launch the Streamlit dashboard:
   ```bash
   streamlit run app.py
   ```

## ⚠️ Limitations
- **Demographic Bias:** Social media users do not represent the entirety of Brockton's demographic.
- **Sentiment Extremes:** People are more likely to post online when highly satisfied or upset, often minimizing neutral opinions.
